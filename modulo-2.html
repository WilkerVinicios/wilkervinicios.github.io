Módulo 2

Registro de "Vendas"

Descrição

Registro de Vendas.

  • Desenvolver um programa em JAVA, com interface gráfica e armazenamento dos dados em banco de dados (SGBD PostgreSQL), que permita fazer o controle de vendas em uma loja fictícia.

Programação Orientada a Objetos

Prof: Willians Santos.

  • Programa em java, com interface gráfica.

Estatística Aplicada

Profª: Fabiola de Carvalho Teixeira.

  • O programa deve mostrar, tabela de distribuição de frequência, calcular e mostrar a média, moda, mediana, variância e desvio padrão, alem de mostrar o gráfico a partir da tabela.

Projeto de Banco de Dados

Profª: Alice Mota Faleiro.

  • Fazer a modelagem do banco de dados que será usado para armazenar os dados do programa descrito.

Engenharia de Requisitos

Prof: Francisco Calaça Xavier.

  • Fazer o diagrama de Casos de Uso, para o programa.

Controle de Versão

Prof: Marcelo Almdeida Gonzaga.

  • Publicar e acompanhar as versões dos fontes no Github.
