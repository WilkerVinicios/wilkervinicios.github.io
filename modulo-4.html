Módulo 4

Aplicação Mobile nativa com Web Service.

Descrição

Aplicação Mobile nativa com Web Service.

  • O projeto consistirá em criar um sistema para gerar relatórios sobre o consumo de combustível de veículos de diferentes usuários.

Desenvolvimento de Aplicações Moveis

Prof: Alex Rabelo Ferreira.

  • Criação de um projeto Android com SDK 29 e Mínimo SDK 16
  • Criação de todas telas do protótipo.
  • Criação de banco de dados local como cache do web service. O cache será criado a primeira vez e será atualizado sempre que o aplicativo reiniciar.
  • O aplicativo irá cadastrar os veículos e consumos tanto no banco de dados quanto na API do webservice. O usuário será salvo somente no webService e como SharedPreferences no dispositivo quando logado.
  • O aplicativo deve avisar quando não tiver acesso à internet.

Entrega Continua

Prof: Alex Rabelo Ferreira.

  • O projeto do aplicativo deverá ser feito utilizando um repositório público do Git
  • Implementar um pipeline no servidor Jenkins do Senac para acessar o git e fazer checkout do projeto do aplicativo.
  • O pipeline deverá conter as etapas: execução de testes, build e geração de APK assinado.
  • Entregar o arquivo Jenkins que descreve todo o pipeline. Lembrando que deverá ser executado em ambiente linux, portanto, somente comandos ShellScript que irão ser aceitos.

Empreendedorismo

Prof: Itair Pereira da Silva.

  • Elaborar um plano Canvas contendo os seguintes itens: parcerias chave, atividades chave, recursos chave e proposta de valor, relacionamento com o consumidor, canais, segmentos de clientes, custos e fontes de receita.

Ferramentas para Engenharia de Software

Prof: Braully Rocha da Silva.

  • Configurar um servidor Jenkins para entrega contínua, com os plugins do git, maven, gradle e junit. Exportar o backup da configuração realizada e entregar esse arquivo como parte do projeto.

Motodologia de Desenvolvimento Ageis

Prof: Andre Luiz Esperidião.

  • Gerenciar o projeto de desenvolvimento utilizando-se do framework Agile Scrum.

Validação e Verificação de Softwares.

Profª:Ana Flavia Marinho de Lima Garrote.

  • Desenvolver os casos de teste dos requisitos funcionais do sistema, realizar os testes e elaborar o relatório de teste.

Web Services

Prof: Marcelo Faustino Rodrigues.

  • Desenvolver a API Rest em Java.
